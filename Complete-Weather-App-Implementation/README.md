# 🌤️ Weather App

A modular, production-ready weather application that fetches real-time weather data from OpenWeatherMap API.

## Features

- ✅ Clean modular architecture (API client, parser, cache, main)
- ✅ Comprehensive error handling (network, invalid city, rate limits)
- ✅ JSON parsing with validation
- ✅ Smart caching (reduces API calls)
- ✅ User-friendly CLI with emoji icons
- ✅ Unit tests included

## Setup

### 1. Get an API Key
Sign up at [OpenWeatherMap](https://openweathermap.org/api) (free tier works fine)
